21-22 Oct, Online, CFP now opened!


PyCon Sweden 2021 will be held on 21-22 October, online. Call for Proposal is now opened!



Submit a Proposal!
INTRO

Call for Proposal (CfP)

We are pleased to announced that PyCon Sweden is returning for the 2021 edition. Due to the uncertain COVID-19 situation in Europe, PyCon SE 2021 will once again be an online event (Completely Free!). We are accepting both regular talks (25 minutes) and workshops (up to 90 minutes). In particlar, we will have five tracks this year, covering a wide range of interests in the community. The conference will follow Stockholm time. We will release more details about participation very soon. Stay tuned to our social media to know more!

To read more about the CFP and submit a Proposal Click Here!

In particular, the areas of interest are as follows:

Important Dates:

Submission deadline: 5 September 2021

Notification of acceptance: 12 September 2021

Conference: 21-22 October 2021

Data Science, AI, and Machine Learning

The popularity of Python among data scientists is one reason why all major frameworks are supporting Python as a binding. At the same time, the opposite is also true. This track is dedicated to talks that concern Data Science, AI, Machine Learning, and related areas. For example: - Python frameworks for Data Analytics, AI, and Machine Learning - Deploying ML models - Computer Vision - Natural Language Processing - Development of data pipelines - Data pre-/post processing - Scalable ML - Time series analysis - Domain-specific case studies - Visuzliation

Scientific and High-Performance Computing

Python is one of the favorite languages that is used by computational scientists and in academia. With reasonable performance and its versatile nature, it is convenient for scientists to prototype code and perform experiments. However, this can reach a bottleneck once it has to be deployed to a large scale. This track is dedicated to talks that concern the acceleration of scientific computing, optimizations, and high-performance computing. For example: - Computational Biology, Chemistry, Physics, and Mathematics - Computational Finance - Accelerated NumPY - Just-In-Time (JIT) compilation of Python code - Accelerating Python code by offloading in C/C++, Cython, Pybind11, etc. - Performance profiling and optimization - Accelerator offloading of Python software: GPU, FPGA - Optimization techniques

Web development, applications, and database technologies

Python is a major powerhouse for web applications. This track is dedicated to talks that concern Python services that support web applications and databases. For example: - Web application frameworks - Database technologies - Authentication systems - Information Retrieval, search engines - Websockets and interactive communications - Web based services (e.g. chatbots)

Education and professional development

Python is now widely used as an introductory language for Computer Science students, and even beyond. This track gathers experiences in the use of Python in education. For example: - Python in the classroom - Python as a first language - Programming for non-computer science students - Developing algorithms and data structures in Python - Career development - Fun with Python

Software Engineering, DevOps, Testing, and Security

Python is a versatile language that can be used anywhere in software development. This track gathers talks that are related to software engineering, DevOps, Testing, and Security. For example: - Testing frameworks and continuous integrations for Python code - Microservice, virtualization, and containerization for Python services - Python syntax semantics - Latest feature in the Python language - Tools - Best practices, refactoring, and case studies - Infrastructure - Design patterns in Python - Game development - IoT
