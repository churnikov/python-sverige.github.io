Django is slow, if you do it wrong.

Live stream: https://youtu.be/qziGxQbwyzA

Abstract: The talk is about pitfalls in django development, we will go through the query optimisation, cacheing (or when to not cache) and how to do it properly, what and how to measure (applies to all python code).

About the speaker: Magnus Knutas has been developing for 20 years as a consultant for both big and small companies, has done code for homeassistant and maintain some libs that are reversed-enginered from "open" api:s.
