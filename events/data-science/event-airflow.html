Don't just go with the flow using Airflow

Live stream: https://youtu.be/IhtqZ0XNYes

Abstract: How to use Airflow beyond plug and play, build it both for developers and customers to achieve a smooth experience. A talk about how we build scalable big data tools at Klarna using Python.

About the speaker: Keven Neville has a background in statistics and machine learning but he works as a data engineer. How did that happen? He applied for a job as a data scientist at Klarna, Klarna said "no you would be a better fit to work more with data and less with machine learning", and now he works as a data engineer in the Data domain at Klarna.
Besides working with data he sees himself a a semi-professional fantasy football player since he almost earned cash doing it.
