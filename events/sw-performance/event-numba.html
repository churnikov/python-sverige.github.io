Accelerating Scientific Computing using Numba

Live stream: https://youtu.be/d_21Q0UoWrQ

Abstract: Ease of learning, usability & vast package ecosystem are some reasons for the wide adoption of Python. But, as researchers we often encounter scenarios leaving us scratching our heads in search of ways to speed up our code. This talk explores how Numba can accelerate scientific computing using Python.
